# tgict
an opinionated set of dictionaries for TGSpeechBox

This is a dictionary for [TGSpeechBox](https://github.com/tgeczy/TGSpeechBox) created by fastfinge.  Entries may, from time to time, migrate from here into the default TGSpeechBox dictionary.  This dictionary contains propper nouns, brands, fictional characters, obscure words, and other mispronounciations fastfinge encounters. 

While pull requests are accepted, for inclusion into this dictionary, words must:
* be pronounced the way that fastfinge thinks sounds correct
* be a word fastfinge knows how to pronounce

Once the dictionary subscription is available, you are welcome to subscribe to this dictionary.  Fastfinge is a Canadian (from Ontario) who works in technology, and reads a lot of science fiction, fantasy, and litrpg, so expect Canadian-leaning pronouncing, with words from the above fields.
